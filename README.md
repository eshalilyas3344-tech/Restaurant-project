# Bella Vista Restaurant Website

A modern, responsive restaurant website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive Navigation**: Smooth mobile menu with hamburger toggle
- **Modern UI**: Clean, elegant design with Italian restaurant theme
- **Fast Loading**: Optimized for performance with external CDN resources

## Pages

- **Home** (`index.html`): Landing page with hero section, specials, and restaurant info
- **Menu** (`menu.html`): Full menu with categories and pricing
- **Reservations** (`reservations.html`): Table booking form
- **Gallery** (`gallery.html`): Photo gallery of dishes and restaurant
- **About** (`about.html`): Restaurant story and team information
- **Contact** (`contact.html`): Contact form, location, and hours

## Technologies Used

- HTML5
- CSS3 (Custom styles in `restaurant-style.css`)
- JavaScript (Interactive features in `restaurant-scripts.js`)
- Font Awesome (Icons)
- Google Fonts (Typography)

## Deployment

This website is ready for deployment on:

### GitHub Pages
1. Push this repository to GitHub
2. Go to Settings > Pages
3. Select "Deploy from a branch"
4. Choose "main" branch and "/ (root)" folder
5. Click Save

### Other Platforms
- Netlify: Drag and drop the folder or connect your GitHub repo
- Vercel: Import your GitHub repository
- Any web server that supports static files

## Local Development

To run locally:
1. Clone the repository
2. Open `index.html` in your web browser
3. No build process required - it's pure HTML/CSS/JS
